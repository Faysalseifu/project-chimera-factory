# Agentic Telemetry – Visible Signals

This document lists the user‑visible signals expected in the dashboard and in logs.

## Visible Signals
- Agent heartbeat and last action timestamp.
- Success signals per platform (likes, replies, shares).
- Risk flags and escalations.
- Budget alerts and spend anomalies.

## Data Sources
- MCP tool call logs.
- Judge decision stream.
- Post‑publish analytics ingestion.
